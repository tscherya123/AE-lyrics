# AE-lyrics

Генератор субтитрів SRT із синхронізацією тексту пісні на базі Faster-Whisper.

## Особливості

- підтримка ручного введення тексту або автоматичного розпізнавання (через Faster-Whisper);
- побудова таймінгів на основі фактичного аудіо завдяки словарним таймкодам Whisper;
- повторне використання вже розпізнаних таймінгів для пришвидшення наступних генерацій;
- ручне редагування, копіювання та експорт SRT.

## Підготовка до точного вирівнювання

1. Встановіть залежності:

   ```bash
   pip install faster-whisper python-levenshtein
   ```

2. За замовчуванням використовується модель `medium` та пристрій `cpu`. За потреби можна налаштувати
   змінні середовища:

   - `WHISPER_MODEL_SIZE` — назва моделі (наприклад, `small`, `large-v3`).
   - `WHISPER_DEVICE` — `cpu`, `cuda` або `auto`.
   - `WHISPER_COMPUTE_TYPE` — тип обчислень (`float32`, `float16`, `int8_float16` тощо).
   - `WHISPER_LANGUAGE` — код мови (за замовчуванням `uk`).

Після цього програма зможе автоматично вирівнювати кожен рядок тексту відповідно до реальних таймінгів аудіо.
