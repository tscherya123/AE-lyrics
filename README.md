# AE-lyrics

Генератор субтитрів SRT із синхронізацією тексту пісні.

## Особливості

- підтримка ручного введення тексту або автоматичного розпізнавання (через `speech_recognition` + Vosk);
- побудова таймінгів на основі фактичного аудіо при наявності офлайн-моделі Vosk;
- конвертація аудіофайлів до необхідного формату для точного вирівнювання;
- ручне редагування, копіювання та експорт SRT.

## Підготовка до точного вирівнювання

1. Встановіть залежності:

   ```bash
   pip install vosk pydub
   ```

2. Завантажте підхожу Vosk-модель (наприклад, `vosk-model-small-uk-v3`) та розпакуйте її.

3. Встановіть змінну середовища `VOSK_MODEL_PATH`, що вказує на каталог із моделлю:

   ```bash
   export VOSK_MODEL_PATH="/path/to/vosk-model"
   ```

Після цього програма зможе автоматично вирівнювати кожен рядок тексту відповідно до реальних таймінгів аудіо.
